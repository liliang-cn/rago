Getting Started

Quick Installation

Get RAGO v2 up and running in minutes with multiple installation options

Download Pre-built Binary

Download the latest release for your platform:

Terminal
# Linux/Mac wget https://github.com/liliang-cn/rago/releases/latest/download/rago-linux-amd64 chmod +x rago-linux-amd64 ./rago-linux-amd64 serve # Windows # Download from: https://github.com/liliang-cn/rago/releases

Build from Source

Clone the repository and build using Make:

Terminal
git clone https://github.com/liliang-cn/rago.git cd rago make build ./build/rago serve

Run with Docker

Use Docker for containerized deployment:

Terminal
docker run -d \ -p 7127:7127 \ -v ~/.rago:/root/.rago \ liliang/rago:latest serve

Configuration

Create a rago.toml file with your settings:

rago.toml
[server] port = 7127 host = "0.0.0.0" [providers] default_llm = "ollama" default_embedder = "ollama" [providers.ollama] type = "ollama" base_url = "http://localhost:11434" llm_model = "llama3" embedding_model = "nomic-embed-text" [mcp] enabled = true servers_config_path = "mcpServers.json"
